## Project Overview

This project is a (unofficial) sample knowledge base (KB) built using Gitbook--a documentation focused CMS.

This knowledge base has clear information architecture (IA), grouping content into logical categories such as getting started, How-to Guides, FAQs, and Troubleshooting.

The project focuses on clarity, scannability, and ease of navigation, rather than technical implementation.

## Features

- Provides clear KB categories
- Simple step-by-step guidance
- Enables self-service support through searchable, well-structured docs
- Designed end users (non-technical) with simple and conversational language

## Technologies Used

- **Gitbook**--a documentation CMS-- for KB management
- **Markdown**, content authoring format used within GitBook
- **Web-based editor** for content authoring and updates
- Built-in navigation and publishing provided by Gitbook
